# Your On Demand Food Ordering Solutions

A full-stack online food ordering web application built using React.js, Node.js, Express.js, and MongoDB.

## Features
- User Login & Signup
- Browse Restaurants
- Browse Food Items
- Add to Cart
- Place Orders
- Admin Dashboard

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB
- JWT
- HTML
- CSS
- JavaScript

## Installation

### Client
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```
